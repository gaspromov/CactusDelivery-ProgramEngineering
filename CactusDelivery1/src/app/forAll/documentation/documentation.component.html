Техническая документация к данному проекту.

Весь код располагается по данной ссылке

Для того, чтобы запустить данный проект у себя:
1. Скачайте файлы из гит
2. Введите команду в консоле в папке проекта npm i
3. Введите команду в консоле в папке проекта ng serve -o

Файлы каждого статуса юзера хранятся в соответствующей папке:
Админа в Admin, Клиента в User, Доставщика в Carrier, Незалогиненного в Unsign

Все сервисы для работы с бд располагаются в папке shared. В качестве базы данных выступает Firestore из API Firebase.
В сервисе AuthService происходят все процессы связанные с авторизацие(sign-in, sign-up, sign-out и тд);
В сервисе OrderService происходит получение, изменение, удаление и создание заказов. А также получение и изменение данных о пользователе.

Все существующие маршруты прописаны в AppRoutingModule. Там же осущесвляется предоставление и ограничение доступа к некоторым маршрутам с помощью Guards. Все Гарды хранятся в папке Shared


У каждого из пользователей реализован стандартный вывод данных, получаемых с бд. В файле TS, функции ngOnInit каждого компонентов, которые выводят данные мы подписываемся на получение данных из бд и соответсвенно все данные у нас записываются в формате Observable и после выводятся в файл html компонента

В данном проекте используется bootstrap,jquery и Firebase
